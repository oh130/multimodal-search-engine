1. 이미지 검색 미검증
   - H&M 이미지 파일이 없어 image / hybrid 검색을 실제로 테스트하지 못함

2. 검색 성능 지표 미측정
   - MRR ≥ 0.55, NDCG@10 ≥ 0.50 달성 여부 확인 안 됨
   - evaluation/ 스크립트로 테스트셋 기준 측정 필요

3. FAISS 인덱스 타입 검토

   - 성능 요건 미달 시 인덱스 타입 변경 고려

4. Two-Tower Candidate Generation 미구현
   - 현재 인기도 기반 폴백만 동작, 모든 사용자가 cold start 처리됨
   - User Tower / Item Tower 구현 및 FAISS 연동 필요

5. DeepFM Ranking 미구현
   - 스켈레톤(docstring)만 있음
   - CTR/CVR 예측 모델 구현 필요, AUC ≥ 0.70 달성 목표

6. 세션 인코더 미구현
   - GRU 또는 Transformer 기반 세션 인코더 스켈레톤만 있음
   - 장기 선호(User Tower) + 단기 관심(세션) 결합 로직 없음

7. Re-ranking 로직 미구현
   - 동일 카테고리 연속 3개 제한 없음
   - MAB(Epsilon-Greedy/UCB) Exploration 슬롯 없음
   - 신규 상품(등록 7일 이내) 노출 보장 없음

8. 추천 성능 지표 미측정
   - Recall@300, HitRate@50, NDCG@50, AUC, Coverage 전부 미측정
