# Checkpoints

학습된 추천 모델 가중치를 저장하는 위치다.

## 역할
- candidate retrieval 모델과 ranking 모델의 저장 산출물을 관리한다.
- serving 계층이 로드할 파일 경로를 고정하는 기준점 역할을 한다.

## 현재 상태
- 구조만 먼저 맞춘 단계다.
- `two_tower.pt`, `deepfm.pt`는 placeholder 파일로만 존재한다.

## 예정 범위
- 모델 버전별 저장 규칙 정의
- 체크포인트 파일명 규칙 정의
- export 시점 메타데이터 관리 규칙 정의

## 주의사항
- 실제 대용량 가중치 파일은 추후 관리 정책에 따라 다룬다.
- 학습 산출물과 임시 실험 파일이 섞이지 않도록 기준을 분리한다.
